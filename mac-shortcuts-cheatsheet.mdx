


Mac 电脑的快捷键使用时非常有必要的，尽量不用鼠标，可以提高效率。掌握 Mac 的快捷键操作不仅能够大幅提升工作效率，还能让您的操作更加流畅自如。

### 代码编辑器 常用快捷键

**强烈推荐**！为程序员设计的高效快捷键，能极大提升编码效率。



| 快捷键 | 功能描述 | 使用频率 |

| :--- | :--- | :--- |

| **`Command` + `/` (⌘/)** | **切换行注释**：快速注释或取消注释选中的代码行。 | ⭐⭐⭐⭐⭐ |

| **`Option` + `↑`/`↓` (⌥↑/⌥↓)** | **上/下移动行**：将当前行或选中的多行代码整体移动。 | ⭐⭐⭐⭐⭐ |

| **`Shift` + `Option` + `↑`/`↓` (⇧⌥↑/⇧⌥↓)** | **上/下复制行**：将当前行或选中的代码复制并粘贴到上方或下方。 | ⭐⭐⭐⭐ |

| **`Command` + `[`/`]`** | **增/减缩进**：对选中代码块增加或减少一级缩进。 | ⭐⭐⭐⭐ |

| **`Shift` + `Command` + `K` (⇧⌘K)** | **删除整行**：快速删除当前行。 | ⭐⭐ |

| **`Command` + `Option` + `↑`/`↓` (⌘⌥↑/⌘⌥↓)** | **添加多光标**：在相邻行创建多个光标以进行同步编辑。 | ⭐⭐⭐ |

| **`Command` + `Enter` (⌘↩)** | **在下方插入新行**：无论光标在哪，都在当前行下方开始一个新行。 | ⭐⭐⭐ |

| **`Shift` + `Option` + `A` (⇧⌥A)** | **切换块注释**：使用 `/* ... */` 形式进行注释。 | ⭐⭐ |



使用频率说明：

- ⭐⭐⭐⭐⭐ 高频必备（每天都会用到很多次）

- ⭐⭐⭐ 常用（经常会用到）

- ⭐ 偶尔使用（特定场景下会用到）
